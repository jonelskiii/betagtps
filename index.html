Tutorial Windows

Desktop
  1. Langkah pertama: Tekan ⊞ Win + R untuk membuka Run
  2. Langkah kedua: Ketik C:\\Windows\\System32\\drivers\\etc lalu Enter
  3. Langkah ketiga: Klik kanan file hosts → buka dengan Notepad (Admin)
  4. Langkah keempat: Tambahkan host dari tombol Salin IP
  5. Langkah kelima: Simpan file, lalu buka Growtopia!
  6. Salin host: 91.134.85.13 growtopia1.com 91.134.85.13 growtopia2.com 91.134.85.13 www.growtopia1.com 91.134.85.13 www.growtopia2.com

Tutorial Android

Mobile
  1. Langkah satu: Unduh PowerTunnel
  2. Langkah dua: Klik tombol PowerTunnel di atas
  3. Langkah tiga: Buka PowerTunnel → Pengaturan Host
  4. Langkah empat: Tempel URL, ubah periode update host menjadi "saat mulai"
  5. URL akhir: https://gtpshost.com/raw/betagrowtopia

Tutorial iOS

Apple
  1. Langkah satu: Unduh Surge 5
  2. Langkah dua: Pergi ke IMPORT → Unduh Profil dari URL
  3. Langkah tiga: Tempel URL dari tombol di atas
  4. Langkah empat: Atur dan izinkan konfigurasi VPN
  5. URL akhir host: https://gtpshost.com/ios/betagrowtopia
